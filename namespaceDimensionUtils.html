Detailed Description

Includes the DimensionType enum and helper functions for dimension types.

Author
Amartya Das Sharma

Enumeration Type Documentation

◆ DimensionType

enum class DimensionUtils::DimensionType
strong

enum storing the axes and direction.

The dimensions are POSX, NEGX, POSY, NEGY, POSZ and NEGZ.

This is hardcoded for 3D, and ERROR is included for sanity checks.
