inlinestatic

Return an OptFFT_API BasicOptFFT is an API for Kurzak's code, it require a square abstracted matrix (=> input matrix have size_x = 2*size_y) and an order between 5 and 15 FakedOptFFT use the same scheme as a true optFFT but perform the FFT and IFFT with FFTW by reconstructing the whole matrix and using a default FFT alg., it works for any matrix.

Parameters
constint order, order of the expansion
constbool square_matrix, == (size_x == 2*size_y)
Returns
optFFT_API* pointer to an implementation of the OptFFT_API
