Detailed Description

Provides interface for generic implementations of an overlapping Communication and Computation for the balance and exchange part, in conjunction with the cell traversal. This class provides blocking implementations. Derived classes should both implement the doInitBalanceAndExchange method, as well as the performComputation method.
