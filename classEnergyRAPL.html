Detailed Description

This plugin computes the energy consumption of the simulation using RAPL counters via the sysfs interface.

Author
Ruben Horn

The output of this plugin is either written to the info logger or to a tab separated file (.tsv) with the columns milliseconds, simstep and joule every writefrequency simulation steps or once at the end.

NOTE:

  • You must ensure that the files under /sys/class/powercap/intel-rapl/ are readable.
  • The total energy is measured over the package and DRAM domains, if present. (See section 15.10.2 of the Intel Architectures Software Developer’s Manuals Volume 3. The actual domains that are used are printed to the info log.) Energy measurements using RAPL are not representative of the total energy consumption of the system and comparing different systems is not trivially possible. Instead, they should only be used to compare the energy consumption of different applications/settings on the same system. For more practical information on RAPL, see Khan et al. 2018. RAPL is an Intel technology and has limited support on AMD (cf. Schöne et al. 2021).
