Detailed Description

template<typename T>
class CtrlVar< T >

Template class for defining a variable triplet: 'target', 'actual' and 'spread'. This is intended for storing information of a observable like e.g. the temperature in a control region: its target value, actual value and the spread between those.

You can also combine it with the template class CommVar, to store local and global information:

CtrlVar<CommVar<double> > temperature;
